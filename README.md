3. Obiectivele Proiectului

Dezvoltarea unui model de clasificare (ex: Random Forest, SVM sau Neural Networks) care să identifice donatorii probabili.

Optimizarea resurselor: Reducerea costurilor operaționale prin țintirea precisă a donatorilor prin campanii de informare (SMS/Email), evitând contactarea celor care au o probabilitate mică de a se prezenta.

Analiza corelațiilor: Identificarea modului în care "vechimea" unui donator (Time) influențează loialitatea acestuia față de centrul de transfuzii.
